Sublime PICO-8
==============

Thanks for installing the PICO-8 plugin for the [Sublime Text 3](https://www.sublimetext.com/) editor.

Complete Installation
---------------------

Before starting using this package, navigate to the package folder and manually install PICO-8.ttf to your system fonts.

Alternatively, you can also download the font from this link:
https://raw.githubusercontent.com/Neko250/sublime-PICO-8/master/font/PICO-8.ttf
